## Información
Aplicación básica de prueba creada con el fin de contar con una base para impresión térmica con la máquina.

## Dependencias
- com.sunmi:printerlibrary:1.0.18 (Apache 2.0) - [Maven](https://mvnrepository.com/artifact/com.sunmi/printerlibrary/1.0.18)

## Hardware requerido
- Sunmi V2 Pro (Versión normal o custom como PedidosYa)

## Notas
- Demo para impresión básica. App probada en Sunmi V2 Pro, edición de PedidosYa.
